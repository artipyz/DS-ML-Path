# Основны

Для базового ознакомления с вероятностями воспользуемся разделом математикии - *Теория множеств (Set Theory)*

## Термин теория множеств

Теория множеств это раздел математики которая базируется на концепции множеств.

Множества в свою очередь это простой набор данных, вещей и т.д.

Например мы можем использовать множества для того что сказать какие вещи лежат в сумке:
- Книга
- Бумага
- Папка
- Шляпа
- Ручка
- Шоколад

```{'Book', 'Paper', 'Folder', 'Hat', 'Pen', 'Chocolate'}```

## Правила множеств

- В математике множества представляются в виде списка заключенных в фигурные скобки
- Каждый элемент множества является уникальным и не имеет дублей
- Элементы в списке не упорядоченны
- Для обозначение множества используется заглавная буква

Учитывая все правила можно сказать что:

{1,2,3,4,5} = {5,4,3,2,1}

A = {1,2,3,4,5}

- Множества также могут иметь подмножества. Например:

A = {1,2,3}

B = {1,2,3,4,5}

Множества A является подмножеством множества B

## Понятия эксперимент и пространство набора

- В теории вероятности, *эксперимет* это что такое что приводит к наблюдениям с определенным уровнем неопределенности.
- Точка выборки (*sample point*) это единственный возможный результат эксперимента.
- Пространство выборки (*sample space*) это множество всех возможных точек выборки для эксперимента

Например проведем эксперимент с подбрасыванием монеты:

- Монету мы будет подбрасывать дважды подряд и записывать значения
- H - Heads
- T - Tails

Согласно правилам мы можем получить такие значения:
- HH
- TT
- HT
- TH

Запишем значения в набор:

S = {HH, TT, HT, TH}

Допустим нам интересно получить только один результат из возможных вариантов **HH**

Определенный результат или множество результатов называют *событием* (*event*) которое в свою очердь является подмножеством пространства выборки (*Sample Space*)

В данном примере мы могли бы рассмотреть три события:
- A = {HH}
- B = {TT}
- C = {TH, HT}

Частотное определение вероятности звучит так - если мы проведем эксперимент бесконечное кол-во раз, вероятность каждого события будет пропорцией кол-ва возникновений каждого события.

К сожалению мы не сможем подрабрасывать монету бесконечное кол-во раз, но мы можем провести эксперимент достаточно большое кол-во раз например 1000 раз.

Хорошо мы подбросили монету дважды 1000 раз и получили результат:
- {HH}: 252
- {TT}: 247
- {HT}: 256
- {TH}: 245

Для расчета вероятности возникновения события используется формула

P (Event) = (Number of Times Event Occured)/(Total Number of Trials)

Где
- P - обозначение вероятности
- Number of Times Event Occured - кол-во раз которое событие происходило
- Total Number of Trials - общее кол-во раз проводимых экспериментов

Таким образом получаем что вероятность возникновения интересного нам события равна

P (HH) = 252 / 1000 = .252

Таким образом мы получаем 25.2% вероятности возникновения выбранного события

Чтобы быть более уверенным в том что мы близки к истинной вероятности возникновения события, мы можем воспользоваться *законом больших чисел* (**Law of Large Numbers**)

# Law of Large Numbers

Мы не можем проводить эксперимент бесконечное кол-во раз.

Но мы можем проводить эксперимент очень большое кол-во раз, и чем больше экспериментов будет проведенно тем ближе значение вероятности возникновения события будет **истинным**

Это и называется **законом больших чисел** (**Law of Large Numbers**)

## Действия над множествами

Существует несколько действия над множествами
- ***Union*** - объединение - действие объединения, объединяет два множества вне зависимости есть они в каждом множестве или только в одном из них
    - Например у нас есть два множества
        - A = {1,3,5}
        - B = {3,4,5,6}
    - Результатом данного действия будет новое множество которое будет равно
        - A or B = {1,3,4,5,6}
    - Как уже было обозначено объединение множеств *union* можно обозначить словом *or*
- ***Intersection*** - пересечение - действие результатом которого является новое множество со значениями которые есть в обоих множествах
    - Для примера возьмем множества которые были обозначены ранее. Результатом действия пересечения будет
        - A and B = {3,5}
    - Вхождение обозначается словом *and*
- ***Complement*** - не вошедшие в множество. Другими словами если у нас A предстваляет собой множество значений полученные при броске игральной кости то compliment этого множества будет равен
    - A^C = {2,4,6}
    - Обозначает действие как множетсво A в степени C

## Independence and Dependence (Независимость и зависимость)

В теории вероятности существует два понятия такие как
- Independence - независимость - в случае если результат следующего эксперимента не зависит от результата предыдущих экспериментов
    - Например подброс монетки - результат каждого последующего эксперимента не зависит от результатов предыдущих
- Dependence - зависимость - в случае если резульат следующего эксперимента зависит от результата предыдущих экспериментов
    - Например вытягивани карты из колоды - если мы хотим вытянуть туза из колоды (36 карт) при этом каждый раз убирать вытянутую карту то результат каждого следующего эксперимента будет повышаться так как кол-во карт будет становиться меньше

**Почему очень важно знать событие зависимое или не зависимое? Это значительно помогает более точно определить вероятность возникновения события.**

## Mutually Exclusive Event (Взаимоисключающие события)

Два события называют взаимосключающими если они не могут произойти одновременно

Например при броске монеты мы не можем получить и Орел и Решку одновременно поэтому такое событие называют взаимоисключащим

А если мы бросаем кость и проверяем два события
- значение игральной кости будет не четным
- значени игральной кости будет выше 2

В таком случае данные события **НЕ** взаимоисключающие. И будут иметь пересечением множество значений {3,5}

Другими словами можно определить взаимоисключающие события по их пересечению
- Взаимоисключающие события имеют пустое пересечение
- Не взаимоисключающие имеют множество в результате пересечения
